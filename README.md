# Star Wars Data Explorer

## Project Overview
The Star Wars Data Explorer is a web application designed to allow users to interactively explore various categories of data from the Star Wars universe, including characters, films, and starships. This application demonstrates fetching data from the Star Wars API (SWAPI) using both the Fetch API and Axios, presenting this data in a visually appealing card format.

## Features
- **Interactive Data Exploration**: Users can select which category of Star Wars data they wish to explore using interactive buttons.
- **Dynamic Data Display**: The application fetches data dynamically and displays it in card format, including images, titles, and key details.
- **Error Handling**: Implements robust error handling to provide feedback on data fetching issues.

## Technologies Used
- **HTML/CSS**: For layout and styling.
- **JavaScript**: For logic and API interactions.
- **Fetch API and Axios**: For making HTTP requests.
- **SWAPI (Star Wars API)**: Provides the data used in the project.
